شرح لتنسيق العناصر باستخدام CSS

1. الخلفية (background-color & background-image)

A. الخصائص:

1- background-color: تحديد لون الخلفية

2- background-image: إضافة صورة خلفية (url("مسار/الصورة.jpg"))

3- background-repeat: التحكم في التكرار (repeat, no-repeat, repeat-x, repeat-y)

4- background-position: تحديد الموضع (top, bottom, left, right, center, x% y%)

5- background-size: التحكم في الحجم (cover, contain, قيم رقمية)


B. مثال:

#bodyy {

background-color:rgb(84, 44, 163);

background-image: url("../../Img/backg.jpg");

background-repeat: no-repeat;

background-size: cover;

background-attachment: fixed;

}

2. النص (text):

A. الخصائص:

1- text-align: المحاذاة (left, right, center, justify)

2- text-decoration: التزيين (underline, overline, line-through, none)

3- text-transform: التحويل (uppercase, lowercase, capitalize)

4- letter-spacing: تباعد الحروف (قيم مثل 2px, -1px)

5- line-height: ارتفاع السطر (بدون وحدات أو قيم مثل 1.6)

6- font-weight: bold; تضخيم النص


B. مثال:

.tt {

text-align: center;

line-height: 0.5;

text-decoration: none;

font-size: small;

font-weight: bold;

3. الخط (font):

الخصائص:

1- font-family: عائلة الخط ("Arial", "Helvetica", 'cairo')

2- font-size: حجم الخط (16px, 1.2rem, 120%)

3- font-weight: السمك (normal, bold, 700)


B. مثال:

ffoun {

font-family: 'cairo';

font-size: 16px;

font-weight: normal;

}

4. الألوان (color):

أنظمة الألوان:

1- HEX: #ffffff (أبيض)

2- RGB: rgb(255, 255, 255) RGB

3- RGBA: rgb(0, 0, 0, 0.5) (شفافية 50%)


مثال :

.lis {

color: red;

background-color: rgb(20%, 100%, 50%, 0.8)

border: 2px solid rgp(0, 0, 0)

}

5. الحدود (border):

الخصائص:

1- border-width: سمك الحدود

2- border-style: النمط (solid, dotted, dashed, double)

3- border-color: لون الحدود

4- border-radius: تدوير الزوايا

5- border-collapse: دمج حدود الجداول

مثال :

.boxx{

border: 1px solid rgb(8, 122, 135);

border: 6px solid blue;

border-radius: 15px 0 15px 0;

}

6. الحشو الداخلي (padding):

المفهوم: المساحة بين المحتوى والحدود

طرق التحديد:

padding: 10px; (كل الجهات)

padding: 10px 20px; (عمودي/أفقي)

padding: 10px 20px 14px 20px; (أعلى، يمين، أسفل، يسار)

7. الهوامش الخارجية (margin):

المفهوم: المساحة خارج الحدود (بين العناصر)

خصائص مهمة:

1- margin: auto; (التوسيط الأفقي)

2- margin-collapse: اندماج الهوامش الرأسية

3- القيم السلبية (margin: -10px;)
